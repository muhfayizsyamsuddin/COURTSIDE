# COURTSIDE
Courtside adalah aplikasi manajemen dan booking lapangan olahraga berbasis web yang bersifat eksklusif (SaaS - Single Tenant) untuk satu instansi atau kompleks olahraga.  Aplikasi ini tidak dikustomisasi khusus untuk satu pemilik bisnis olahraga yang memiliki beberapa lapangan dalam satu kategori olahraga spesifik.
